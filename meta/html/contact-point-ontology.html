Contact point ontology

Classes · Object properties · Data properties · Namespaces · Depiction

Classes

Class Contact point

One or more means of contact for a contactable thing, for example: a contact point having a postal address.

Class Contactable thing

A thing that can be contacted. This is a group or a person in the context of their role in or their membership of a group.

Class Online account

A online account, for example: an online account for the UK Parliament on X.

Class Online account provider

A provider of online accounts, for example: X.

Class Postal address

A postal address.

Object properties

Object property Has contact point

Relates a contactable thing to a contact point. A contactable thing may have one or more contact points. A contact point is for one contactable thing.

ContactableThing — Has contact point — ContactPoint

Object property Has online account

Relates a contact point to an online account. A contact point may have one or more online accounts, or none.

ContactPoint — Has online account — OnlineAccount

Object property Has postal address

Relates a contact point to its postal address. A contact point has one or more postal addresses or none.

ContactPoint — Has postal address — PostalAddress

Object property Has provider

Relates an online account to the provider of that account. An online account has one provider.

OnlineAccount — Has provider — OnlineAccountProvider

Data properties

Data property Account name

Relates an online account to the name of that account. For example: UK Parliament having an account on X with the name @ukparliament.

OnlineAccount — Account name — Literal

Data property Account URL

Relates an online account to the URL of that account. For example: UK Parliament having an account on X with the URL https://x.com/ukparliament.

OnlineAccount — Account URL — Literal

Data property Address

Relates a postal address to the part of its address before its postcode. For example: '35-39 Market Street, Chorley'.

PostalAddress — Address — langString

Data property Availability note

Relates a contact point to a note describing the availability of the contactable thing at that contact point. For example: a constituency office being contactable in working hours only.

ContactPoint — Availability note — langString

Data property Contact form URL

Relates a contact point to the URL of a contact form.

ContactPoint — Contact form URL — Literal

Data property Email address

Relates a contact point to its email address, for example: 'lindsay.hoyle.mp@parliament.uk'.

ContactPoint — Email address — Literal

Data property Fax number

Relates a contact point to its fax number.

ContactPoint — Fax number — Literal

Data property Homepage URL

Relates a contact point to an online homepage.

ContactPoint — Homepage URL — Literal

Data property Phone number

Relates a contact point to its phone number, for example: '01257 271555'.

ContactPoint — Phone number — Literal

Data property Postcode
