Paper type ontology

A model describing the relationships between types of papers.

Classes · Object properties · Subclasses · Namespaces · Depiction

Classes

Class Bill expression

An expression of a bill work at a point in time. A new expression of the bill work is created by any amendment being made to the bill, or when a bill passes from the first House to the second House.

Superclasses · PaperExpression

Class Bill work

A proposal for a new Act of Parliament.

Class Enabled thing

A thing which is intended to exercise a power set out in statute.

Superclasses · PaperExpression

Class Explanatory memorandum expression

A paper intended to explain the purpose, policy objectives and policy implications of an instrument being subject to parliamentary procedure.

Superclasses · PaperExpression

Class Explanatory note expression

A paper intended to explain the purpose of a bill.

Superclasses · PaperExpression

Class Impact assessment expression

A paper, provided by the government, intended to set out the objectives of policy proposals and the costs, benefits and risks of different ways of achieving those objectives.

Superclasses · PaperExpression

Class Paper expression

A paper made available to a House, being an expression of a paper work.

Class Treaty expression

A written agreement between states - or between states and international organisations - which is binding under international law.
