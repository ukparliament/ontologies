Related item ontology

Provides a means to link from a modelled thing to either another modelled thing or a thing at a URI by means of a typed relationship.

Classes · Object properties · Data properties · Namespaces · Depiction

Classes

Class DocumentType

The type of the document being cited, for example: a standing order or legislation.

Class Relationship

A relationship from a modelled thing to either another modelled thing or a document at a URI.

Class Relationship type

The type of a relationship, for example: cites, announces, corrects or commits to.

Class Thing

A thing.

Class Web document

A document available on the web.
