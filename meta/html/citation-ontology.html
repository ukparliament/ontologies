Citation ontology

A model describing the relationship between a thing having a citation and the thing or URI being cited.

Classes · Object properties · Data properties · Namespaces · Depiction

Classes

Class Citation

A citation from a thing to either a thing or a document at a URI.

Class Cited thing

A thing being cited which is an instance of a class in the Parliament data store, for example: a standing order or a Speaker's ruling.

Class Citing thing

A thing citing another thing.

Class DocumentType

The type of the document being cited, for example: a standing order or a section of legislation.

Object properties

Object property Has citation

Relates a citing thing to a citation. A citing thing includes one or more citations. A citation is included in one citing thing.

CitingThing — Has citation — Citation

Object property Has document type

Relates a citation to the document type of the thing at the URI being cited. A citation has one document type, or none.

Citation — Has document type — DocumentType

Object property Refers to

Relates a citation to the thing being cited. A cited thing may have one or more citations. A citation is to one cited thing.

Citation — Refers to — CitedThing

Data properties

Data property Has URI

Relates a citation to the URI of the thing being cited.

Citation — Has URI — Literal
