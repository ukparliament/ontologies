Procedure step ontology

A model describing annotations added to event steps in the procedure model by the Indexing and Data Management Section of the House of Commons Library. Annotations are used to describe how an event step is actualised.

Classes · Object properties · Data properties · Namespaces · Depiction

Classes

Class Event step

A procedural step of type event, being the notion of a event happening in the House of Commons, the House of Lords, bicamerally, in a devolved legislature or external to Parliament and the devolved legislatures.

Class Publication

A publication.

Object properties

Object property Actualise with

Relates an event step to an event step where steps may be actualised together.

Object property Source

Relates an event step to the publication which is the source of information for that event actualising the event step.

Data property Date note

A note that describes the date to use when actualising an event step.
