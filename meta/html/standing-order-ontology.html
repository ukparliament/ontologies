Standing order ontology

A model describing standing orders of legislatures and changes to standing orders over time.

Classes · Object properties · Data properties · Namespaces · Depiction

Classes

Class Business extent

The extent of business covered by a set of standing orders being, for example: public business or private business.

Class Fragment

A fragment of a standing order with its associated intent, as a single intellectual work, as defined in the Functional Requirements for Bibliographic Records specification.

Class Fragment version

A version of a fragment of a standing order, contained by a revision set.

Class House

A parliamentary House, for example: the House of Lords.

Class Legislature

A legislature, being either the UK Parliament or a devolved legislature.

Class Order

A standing order with its associated intent, as a single intellectual work, as defined in the Functional Requirements for Bibliographic Records specification.

Class Order version

A version of a standing order contained by a section of a revision set.

Class Revision

A change to the text of a standing order fragment.

Class Revision set

A set of standing orders, as they exist at the time of revision of one or more of those orders.

Class Revision set section

A grouping of orders under the same title in a revision set.

Class Suspension period

A suspension period is the time during which one or more standing orders are suspended. A suspension period has a start date and, where specified, an end date. Some suspensions do not specify an end date, for example: the suspension of House of Lords Standing Order 10(6) (Hereditary peers: by-elections) pending a further report from the Procedure and Privileges Committee. An order may be 'suspended, for a period' or 'dispensed with, for a purpose'. The suspension period class covers suspensions, not dispensations.
