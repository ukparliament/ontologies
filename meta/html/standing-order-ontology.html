Standing order ontology

A model describing standing orders for both Houses of Parliament, establishing persistent identification of orders and fragments of orders.

An order is an FRBR-style work level item, wrapping different manifestations - described as versions - over time. It represents a platonic entity and associated intent.

Class Order version

A version of an order contained by a section of a revision set.

Class Revision

Class Revision set

A set of sections, containing orders, as they exist at the time of revision of one or more orders.

Class Revision set section

A grouping of orders in a revision set.

Class Suspension period

A suspension period is the time during which one or more standing orders, or fragments of standing orders, are suspended. A suspension period has a start date and, where specified, an end date. These dates are set out in the utility model. Some suspensions do not specify an end date, for example: the suspension of House of Lords Standing Order 10(6) (Hereditary peers: by-elections) pending a further report from the Procedure and Privileges Committee. An order or fragment may be 'suspended, for a period' or 'dispensed with, for a purpose'. Orders and fragments may be dispensed with on a case by case basis: in the House of Commons this is usually indicated by the presence of 'notwithstanding' in the text of the motion. The suspension period class only covers suspensions and does not cover cases where standing orders are dispensed with. The procedure maps have steps which express the dispensing of standing orders.
